6) Complementarios 1

Visor de Fusion 360

En este post veremos como insertar un visor iFrame de Fusion 360 a nuestro sitio web. El modelo utilizado corresponde al molde diseñado para el examen.

Paso 1:

Una vez listo el modelo procedemos a guardarlo en la nube en Admin proyect.

Paso 2:

Ingresar a Autodesk en http://www.myhub.autodesk360.com

Proceder a iniciar sesion con nuestros datos.

Paso 3:

Abrimos el modelo que queremos usar. Para compartir vamos a el signo de conexiones en la parte superior derecha.

Paso 4:

Luego de la opción share, ponemos Embed y seleccionamos el link.

Paso 5:

Finalmente en visual studio code, ponemos el link según lenguaje markdown. Y aparecera el visor de Fusion 360 con nuestro modelo.
